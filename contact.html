HTML Forms

It is used to collect user input.This user input goes to server.

Form element

It acts as container for input elements(text field , radio button , checkbox , submit button

Input element

It is used with help of type attributes(text , radio ,checkbox , submit , button)

Name attribute

It is used to reference the form-data after submitting the form or to reference the element

Label element

It defines label for elements

Action element

The action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.
